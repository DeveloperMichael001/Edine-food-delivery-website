# Flavour Signup Form - Design Exploration

## Selected Design Approach: Modern Minimalist with Vibrant Accent

### Design Philosophy
A clean, contemporary interface that prioritizes clarity and user focus. The design leverages the bold red/orange brand color as the primary accent, creating visual hierarchy through strategic color placement rather than excessive decoration. The layout is centered and spacious, guiding the user's attention naturally through the form.

### Core Principles
1. **Clarity First**: Minimal visual noise, generous whitespace, and clear information hierarchy
2. **Brand-Driven**: The vibrant red/orange (#E74C3C or similar) is the hero color, used strategically for CTAs and key elements
3. **Accessibility**: High contrast text, clear focus states, and readable typography
4. **Progressive Disclosure**: Form elements appear in logical order with clear visual separation

### Color Philosophy
- **Background**: Clean white (`#FFFFFF`) to maximize readability and create a premium feel
- **Primary Accent**: Vibrant red-orange (`#E74C3C`) for buttons, checkboxes, and interactive elements—energetic and action-oriented
- **Text**: Dark charcoal (`#2C3E50`) for body text, lighter gray (`#95A5A6`) for secondary text
- **Borders**: Subtle light gray (`#ECF0F1`) for input fields and dividers
- **Emotional Intent**: The red conveys energy, trust, and action; white conveys simplicity and clarity

### Layout Paradigm
- **Centered Card Layout**: All content contained within a centered, card-like container (max-width: 500px)
- **Vertical Flow**: Single-column layout with clear vertical progression
- **Breathing Room**: Generous padding and spacing between form elements (24-32px)
- **Asymmetric Accents**: Checkbox and button use the red accent to break up the neutral palette

### Signature Elements
1. **Bold Red Checkbox**: Custom checkbox with the brand red color—becomes a visual anchor
2. **Full-Width Red Button**: Prominent CTA with rounded corners and white text
3. **Subtle Input Styling**: Light gray borders, rounded corners, clear focus states

### Interaction Philosophy
- **Hover States**: Subtle background color changes and shadow elevation on interactive elements
- **Focus States**: Clear ring focus on all inputs for keyboard navigation
- **Smooth Transitions**: 200-300ms transitions on color and shadow changes
- **Feedback**: Button press animation with slight scale reduction

### Animation
- **Form Entrance**: Subtle fade-in and slide-up animation for the entire form (300ms)
- **Input Focus**: Smooth border color transition and shadow expansion (200ms)
- **Button Hover**: Slight shadow elevation and background color deepening (150ms)
- **Button Click**: Brief scale-down animation (100ms) for tactile feedback
- **Password Toggle**: Smooth icon rotation (180ms) when toggling visibility

### Typography System
- **Heading (Welcome Back)**: Bold, large display font (36-40px), dark charcoal
- **Subheading (Create Account)**: Medium weight, gray, 14-16px
- **Labels**: Small, medium weight, dark charcoal, 12-14px
- **Input Text**: Regular weight, dark charcoal, 16px
- **Button Text**: Bold, white, 16px
- **Link (Login)**: Regular weight, red accent, underlined on hover

### Font Pairing
- **Display/Headings**: System fonts or bold sans-serif (e.g., -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto)
- **Body/Inputs**: Clean sans-serif for readability (same system font stack)
